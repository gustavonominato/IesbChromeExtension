Política de Privacidade

Extensão: IESB Chrome Extension

Responsável: Nominato Tecnologia

1. Dados coletados

Esta extensão não coleta, transmite, vende ou compartilha dados pessoais dos usuários.

2. Processamento local

A extensão processa localmente, no navegador do usuário, informações exibidas em páginas autorizadas, como dados de notas, frequência e relatórios acadêmicos selecionados pelo próprio usuário.

3. Armazenamento

Alguns dados processados podem ser armazenados temporariamente no armazenamento local do navegador por meio de chrome.storage.local, exclusivamente para permitir que o usuário aplique posteriormente as informações no sistema acadêmico.

4. Compartilhamento com terceiros

Nenhuma informação processada pela extensão é enviada para servidores externos, serviços de terceiros, sistemas de análise, publicidade ou rastreamento.

5. Permissões utilizadas

A extensão utiliza permissões do Chrome apenas para executar suas funções principais, como acessar páginas autorizadas, processar downloads iniciados pelo usuário e armazenar dados localmente no navegador.

6. Segurança

Todo o processamento ocorre localmente no navegador do usuário. A extensão não utiliza código remoto e não executa scripts externos ao pacote enviado à Chrome Web Store.

7. Contato

Para dúvidas sobre esta política, entre em contato com Nominato Tecnologia.
