# Plugin Chrome - Automatiza preenchimento de dados pelos professores

## Objetivo

1. Capturar dados de notas e trabalhos da P1 em uma página, somar os valores e aplicar o resultado no Aluno Online.
2. Capturar dados de notas e trabalhos da P2 em uma página, somar os valores e aplicar o resultado no Aluno Online.

## Como instalar no Chrome

1. Acesse:

chrome://extensions/

2. Ative o modo desenvolvedor.

3. Clique em "Carregar sem compactação".

4. Selecione a pasta do projeto:

meu-plugin-chrome/

5. Acesse o site de captura.

6. Clique com botão direito na página e selecione "Capturar".

7. Acesse o site de aplicação.

8. Clique com botão direito na página e selecione "Aplicar".
